# Prison Infrastructure OS

A premium, Apple-inspired catalog page showcasing three prison-focused digital products:

- India Prison Portal
- AI Prison Management System
- Nyay Setu Justice Support

## Local preview

Run:

```bash
python3 -m http.server 4180 --bind 127.0.0.1
```

Then open `http://127.0.0.1:4180`.

## Deployment

This project is a static site and is ready to deploy directly on Netlify.
